## 👥 프로젝트 한 줄 설명
IoT와 유니티로 체험하는 소방 훈련 시뮬레이션 웹
  
  <br>
  <br>
  <br>
  
  
## 🧑🏻‍🦯 프로젝트 목적
  소방 안전 교육을 목표로 하는 화재 상황 시뮬레이션 웹입니다. 소화기를 IOT로 제작하여 교육 대상자가 소화기 사용법을 익힐 수 있습니다. 캠으로 사용자의 움직임을 분석하여 화재 상황에서 출구를 찾아 대피할 수 있도록 합니다. 더 많은 사람이 올바른 소방 훈련을 받을 수 있게 하는 것이 프로젝트의 목표입니다.


 
 
  <br>
  <br>
 
## 🛠 기술 스택
`공통` - OAuth 2.0

`Front` - React, TypeScript, React-Query, tailwind, npm, Recoil, vite, Unity, C#

`Back` - Spring Boot, JPA, JWT, Swagger

`DB` - MySQL

### 아이디어 차별성
 안전교육을 더 유익하게 하기 위한 노력은 계속되고 있지만 현재의 소방 안전 훈련 방식은 아쉬운 점이 있습니다.  화재 상황 재현을 위해 불을 사용하는 위험성, 소방 대원이 함께해야 하는 인력 낭비 문제가 있습니다. 또한, 소화기를 체험하는 것 역시 소수의 대표만 가능한 형태로 제한되어 있어서 다수의 사람들이 실제 화재 상황에서 어떻게 대응해야 하는지 체험하지 못하고 있습니다. 우리의 프로젝트는 이러한 불편함을 해소하고, 더 많은 사람이 소방 안전 훈련을 직접 체험할 수 있는 시뮬레이션을 제공할 것입니다.

  <br>
  <br>

### 고도화 방안
실제 훈련과 비교해 장비나 재료비에 있어 이점을 가지기 때문에 기존의 화재 훈련을 효율적으로 대체할 수 있을 것으로 보입니다. 또한 재난 상황을 점점 더 추가해 나감으로서 사용자가 좀 더 다양한 상황에 대처할 수 있게 하고 훈련 인증을 제공함에 따라 재난 교육 인증 플랫폼으로 발전할 수 있습니다.
